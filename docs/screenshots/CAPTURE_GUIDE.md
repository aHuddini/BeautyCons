# Screenshot Capture Guide for BeautyCons README

## Recommended Screenshots

Capture these in Playnite Desktop mode with a dark theme for best contrast.
Use Windows Snipping Tool (Win+Shift+S) or ShareX for clean captures.

### 1. Hero Preview (`preview.png`)
- Show the detail view with a game selected that has a colorful icon
- Glow + shimmer visible, mid-shimmer-cycle for the shine bar effect
- Suggested preset: **Huddini - Synthwave Cruiser (Square)**
- Size: ~800x400, cropped to just the icon area + glow

### 2. Glow Styles Grid (`glow-styles.png`)
- If possible, show 4 styles side by side (Neon, Bloom, Star, Halo)
- Or capture each separately and compose in an image editor
- Size: ~600x200

### 3. Shimmer in Action (`shimmer.gif`)
- Animated GIF showing one full shimmer cycle
- Use ScreenToGif or ShareX GIF mode
- Keep short: 3-5 seconds, 15fps, cropped tight to the icon
- Suggested preset: **Golden Foil** or **Chrome**

### 4. Metallic Luster (`luster.gif`)
- Show the directional lighting sweep (lit side bright, shadow side dim)
- Use **Gold** shine style for most dramatic effect
- 3-5 second GIF

### 5. Preset Comparison (`presets.png`)
- Show the Presets tab in the settings panel
- Clean screenshot of the tab layout

### 6. Settings Panel (`settings.png`)
- Show the Icon Glow tab with various settings visible
- Scroll to show the key sections

### 7. Circular vs Square (`effect-shapes.png`)
- Side by side if possible: one icon in Square mode, one in Circular
- Show shimmer effect on both

## File Naming
- Use lowercase with hyphens: `golden-foil.gif`, `settings-panel.png`
- GIFs for animations, PNGs for static shots
- Optimize PNGs with a tool like pngquant or TinyPNG before committing
